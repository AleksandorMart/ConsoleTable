# Person Data Table Viewer

Python-скрипт для чтения tab-разделённого файла с данными людей и отображения их в виде отформатированной таблицы в консоли.

## ✅ Выполненные требования

- [x] Вывод таблицы в консоль с корректным отображением данных
- [x] Заголовок таблицы
- [x] Заголовки столбцов
- [x] Ровные столбцы с разделителем `|`
- [x] Единый формат даты для всех записей
- [x] Формат даты: `YYYY-MM-DD HH:MM:SS`
- [x] Кроссплатформенность (Linux/Windows)
- [x] Учёт ширины консоли с автоматическим сокращением данных
- [x] Поддержка любой кодировки входного файла
- [x] Потоковая обработка файлов до 1GB+
- [x] Именованный параметр `-i`/`--input` для пути к файлу
- [x] Поддержка URL в параметре `--input`
- [x] Покрытие тестами (pytest)

## 🚀 Установка

```bash
# Клонируйте репозиторий
git clone https://github.com/AleksandorMart/ConsoleTable TestProgram
cd TestProgram

# Установите зависимости
pip install -r requirements.txt
```

## 💻 ИспользованиеE

```bash
# Локальный файл
python script.py -i data.txt
python script.py --input /absolute/path/to/data.txt

# URL
python script.py -i https://gitlab.com/toptenov/automation_qa_intern_2026/-/raw/main/data.txt
```

# 📋 Формат входного файла
Файл должен содержать tab-разделённые колонки:
```
ФИО<tab>возраст<tab>адрес<tab>дата
```

Пример:
```
Иванов Иван Иванович	35	г. Москва, ул. Ленина, д. 1	2026-03-15
Петров Петр Петрович	42	г. СПб, Невский пр., 10	2026-15-03
```

# 📅 Обработка дат

- Все даты приводятся к формату YYYY-MM-DD HH:MM:SS
- Если значение месяца > 12, день и месяц автоматически меняются местами
- Некорректные даты отображаются как 0000-00-00 00:00:00

# 🖥️ Отображение в консоли

- Таблица адаптируется под ширину терминала
- При нехватке места содержимое ячеек сокращается с добавлением ...
- Минимальная ширина столбца: 10 символов

# 🧪 Запуск тестов

```bash
pytest test_script.py -v
```

# ⚠️ Важно
- Файл data.txt нельзя изменять (CRC32: 0x2c083a45)
- Не используются готовые библиотеки для отображения таблиц — вся логика форматирования реализована вручную
- Большие файлы обрабатываются построчно для экономии памяти
